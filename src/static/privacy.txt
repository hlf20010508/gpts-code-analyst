Code Analyst Privacy Statement:
This program does not store or remember users' personal data. All data processed by this program, including GitHub repository information and code analysis results, are only available during the current session. After the session ends, this data will not be saved or used for any other purposes.
This program does not request users' personal information. When analyzing GitHub repositories, it only accesses publicly available code and information and will not attempt to access or analyze any private repositories or protected data.
This program maintains the highest standard of security in processing all requests. Any data provided by users during the session will be cleared from this program's system once the session ends.
This program ensures transparent and accurate responses that are limited to the scope of users' requests. It will not infer, store, or use users' data beyond the scope of their posed questions.
